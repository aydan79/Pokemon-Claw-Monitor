## 🧩 What is Pokemon-Claw-Monitor?

Pokemon-Claw-Monitor is a live dashboard that shows the activity of OpenClaw agents using a Pokemon theme. It looks like a pixel-art office where each agent is represented by a Pokemon moving around, sitting at desks, and returning to Pokeballs. This app brings a visual, fun way to track these agents in real time.

The office layout is customizable. You can change the workspace design with a simple configuration file. This app runs on Windows and requires no programming skill to start using it.
